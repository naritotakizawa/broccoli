==============
サンプルゲーム
==============
いくつかのサンプルゲームを公開しています。

サンプルゲームは `pip` ではインストールされないので、実際に動かすためには ``git clone`` を行う必要があります。 ::

    git clone https://github.com/naritotakizawa/broccoli.git
    cd broccoli/samples

roguelike
=============
風来のシレンやトルネコのような、ローグライクゲームのサンプルです。

スクリーンショット
---------------------
画像やGIFのスクリーンショット...


遊んでみる
-----------
::

    cd roguelike
    pipenv install
    pipenv run python main.py


各種キー操作
-------------

:攻撃: z
:上移動: w
:右移動: d
:左移動: a
:下移動: s
